Imagens Dinâmicas

Para criar imagens dinamicas usados a tag <picture> e dentro dela a tag <img> e a tag <source> com alguns atributos como

media
Usado para definir o tamanho maximo a ser considerado para carregar a imagem no atributo srscet
Type
Indica o media type que vai ser usado
Srscet
Vai configurar o nome dá imagem a ser carregado quando o tamanho for atingido
